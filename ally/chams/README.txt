Для создания собственных чамсов необходимо учитывать, что для корректной работы настроек требуется наличие двух материалов: in и out.

В каждом файле ".vmt" обязательно должен присутствовать параметр "$model" со значением "1", иначе материал не будет отображаться.

Рекомендуется для материала in устанавливать параметр "$ignorez" в значение "1", а для материала out — в значение "0".

В данной папке находятся примеры пользовательских чамсов, включая материалы "flat_*.vmt" и "shadow_*.vmt".
